    FAQ — Как работать с этим
  




  

    
Работа с сайтом

    
1. Укажите адреса в виде доменов или IP (поддерживается IPv4 и IPv6).

    
2. Нажмите кнопку Сгенерировать для формирования готового конфига.

    
При необходимости можно скачать YAML-файл.

  


  

    
Что делает кнопка «Упростить»?

    
Кнопка Упростить объединяет домены в правила DOMAIN-SUFFIX, если это возможно (имеет общий основной домен).

    
Это делает конфигурацию компактнее и удобнее для восприятия.

  


  

    
Работа с Clash Verge

    
Полученный конфиг добавьте в Clash Verge через Profiles → Global Extend Config.

    
Правила отобразятся во вкладке Rules и будут применяться автоматически.

  


  

    
Режим прокси

    
В блоке Proxy Mode (Home → Proxy Mode) выберите режим Rule для корректной работы.

  


  

    
Что такое группа?

    
Это имя прокси-группы (например, PROXY или AUTO), через которую будет идти трафик.

    
По умолчанию используется PROXY.

  


  

    
Что такое MATCH?

    
Это правило по умолчанию, применяемое ко всему остальному трафику, не попавшему под другие правила.
